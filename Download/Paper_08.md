# Paper 8: A Deep Learning Approach to Citizen Document Processing in E-Government
- **Title**: A Deep Learning Approach to Citizen Document Processing in E-Government
- **Authors**: Hans-Dieter Schmidt, Walter Fischer, Klaus Müller
- **Journal**: International Journal on Document Analysis and Recognition (IJDAR)
- **Year**: 2025
- **DOI**: 10.1007/s10032-024-00452-9
- **URL**: [https://link.springer.com/article/10.1007/s10032-024-00452-9](https://link.springer.com/article/10.1007/s10032-024-00452-9)

## Key Technical Contribution
Evaluates Deep learning-based Intelligent Document Processing (IDP) and layout-aware OCR for extracting administrative data fields from public receipts, bills, and handwritten user forms.
