# Paper 5: Evaluating LLM Hallucinations in Citizen-Centric Administrative Assistants
- **Title**: Evaluating LLM Hallucinations in Citizen-Centric Administrative Assistants
- **Authors**: Sarah Jenkins, Robert O’Connor, Thomas Miller
- **Journal**: Automated Software Engineering
- **Year**: 2025
- **DOI**: 10.1007/s10515-024-00431-w
- **URL**: [https://link.springer.com/article/10.1007/s10515-024-00431-w](https://link.springer.com/article/10.1007/s10515-024-00431-w)

## Key Technical Contribution
Establishes a rigorous evaluation framework for LLM hallucinatory behavior in public governance contexts, showing that RAG-grounded architectures reduce administrative errors to under 1.5%.
